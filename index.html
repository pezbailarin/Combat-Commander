    Combat Commander RST

    This is a tool to assist players of GMT Combat Commander

    You will never need to find the paper sheet - support tables: Enter nationality, year and dice rolled, and find which units are elegible as reinforcements.

    This app has been made with learning purposes, don't expect it to be a profesional tool. And double check the results if you are serious about gaming :D

    Development evironments used: AIDE in my SGS3 and Eclipse Kepler in my laptop. Here is the repo with the code.

    If you just need the apk please download and open the zip file, and find it inside the bin/ folder. Note you will need to configure your device to allow installation from unknown sources.
